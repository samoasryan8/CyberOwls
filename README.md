# Tramplin (Трамплин) 

**Tramplin** — это совместный проект, разработанный на Java с использованием фреймворка **Spring Boot**. Платформа создана для взаимодействия в IT-сфере: она объединяет тех, кто ищет работу в программировании, и тех, кто предлагает вакансии.

## Технологический стек
*   **Backend:** Java, Spring Boot (Data JPA, Security, Web)
*   **Database:** MySQL 
*   **Build Tool:** Maven 
*   **Frontend:** (Thymeleaf, BootStrap, HTML/CSS)

## Основные возможности
*   **Поиск работы:** Удобный фильтр вакансий для разработчиков разного уровня.
*   **Размещение вакансий:** Компании и рекрутеры могут публиковать предложения о работе.
*   **Личный кабинет:** Профили соискателей и работодателей.
*   **Joint Project:** Проект разработан в команде как масштабируемое решение для IT-рынка.
